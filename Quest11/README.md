## Checklist
* AWS의 VPC는 어떤 서비스일까요?
* 인프라의 네트워크를 가상으로 구축하는 것은 어떤 장점이 있을까요?

* Secrets Manager는 어떤 서비스일까요? AWS 외에 비슷한 역할을 하는 서비스로는 어떤 것이 있을까요?
* 어플리케이션 레벨에서의 보안 전략에는 어떤 것들이 있을까요?
